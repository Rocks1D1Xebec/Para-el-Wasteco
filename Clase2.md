Una clase para entender porque conectar computadores cambia la forma de estudiar, trabajar, comprar, vender y comunicarse


*Socuedad-comercio-conectividad*

comnercio:ventas pagos logistica, atencion

conectividad: redes internet, servicios, dispositivos

Ejemplos reales de conectividad

educacion: aula virtual, videoclases, bibliotecas digitales, entrega de trabajos
comercio: pos/qr, inventarios, pedidos online, atencion por chat

*Comunicacion de datos*

Es el intercambio de daos entre un origen y un destino mediante un medio de transmision, siguiendo reglas acordadas llamadas protocolos.
Idea tecnica: Sin reglas comunes, los dispositivos pueden estar conectados fisicamente pero no entenerse logicamente

*¿que ocurre cuando se envia un dato?*

1. crear el dato:texto imagen solicitud o archivo
2. codificar:convertir en seañles o tramas
3. transportar:usar un medio fisico/inalambrico
4. interpretar: el destino aplica reglas y reconstrye el datos

la red no Aivina: cada paso requiere una regla un formato y una dirrecion para que el dato llegue correctamente

*Vocabulario esencial de comunicacion de datos*

Dato: representacion de informacion texto, numero, umagen, audio, video
Señal: forma fisica/electromagnetica en la que viajan los datos
canal: medio por el cual se transmite la señal
nodo:dispositivio conectado a una red
enlace:conexcion entre nodos
servicio:funcionalidad ofrecida por la red o un servidor

*dos ideas que suelen confundirse*

ancho de banda: capacidad maxima del enlace para tranportar datos
throughput: contidad real de datos entrefada por unidad de tiempo
latencia: tiempo que tarda un datos en ir del origen al desitno
perdida: paquetes que no llegan o deben retranmitirse

*consideraciones sociales de las redes*

Brecha digital: no todoas las personas tienen el mismo acceso a equipos, internet o habilidades digitales
privacidad: la informacion viaja por redes y debe protegerse ante accesos no autorizados
dependencia tecnologica: servivicos criticos pueden detenerse si o no se diseña continuidad y respaldo
uso responsable: evitar abuso de recursos, desinformacion, ataques y practicas inseguras
